# TDG for Software Development

Demo repository for my talk at FOSSASIA Summit 2026.

This repo demonstrates **TDG (Test-Driven Generation)** for building production-grade software using AI Agents with more control, safety, and predictable results.

Modern AI tools can generate code quickly, but real-world software development is more than writing code — it involves maintaining systems, evolving legacy codebases, and ensuring reliability over time.

TDG introduces a structured approach where developers encode their engineering practices into an AI workflow. Instead of random code generation, the AI follows proven development patterns such as TDD, producing safer and more maintainable results.

This repository includes two demonstrations:

• **Replay CLI** — a tool that replays Git commit history step-by-step, making it easier to teach software development and follow the evolution of a codebase.

• **Qoomlee Airline Check-in UI** — a React-based UI demo showing how TDG can be applied to real application development.

The core idea is simple:
**clone developer skills into AI agents and let them work using the same disciplined practices we use.**

This approach helps teams build software faster while maintaining production-grade quality.
